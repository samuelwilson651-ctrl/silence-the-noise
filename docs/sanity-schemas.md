# Sanity CMS Schemas — Silence the Noise™

## Content Types

### article
Fields: title, slug, excerpt, body (portable text), coverImage, author (ref), publishedAt, readingTime, category, pillars[], tags[], featured, memberOnly, seo

### devotional
Fields: title, slug, scripture, scriptureReference, reflection (portable text), prayer, coverImage, publishedAt, category, pillars[], theme, memberOnly, seo

### podcast
Fields: title, slug, episodeNumber, description, audioUrl, duration, transcript, coverImage, publishedAt, category, pillars[], memberOnly

### videoSeries
Fields: title, slug, seriesNumber, description, videoUrl, duration, thumbnailUrl, publishedAt, category, pillars[], memberOnly

### author
Fields: name, credentials, bio, image

### resource
Fields: title, slug, description, resourceType (pdf/guide/worksheet/audio/video/link), fileUrl, externalUrl, coverImage, category, pillars[], memberOnly, comingSoon
